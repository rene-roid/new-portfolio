Scuffed Anime Chess

This is a chess game created in Unity as a personal project. My goal was to deepen my understanding of game development while also learning more about chess. Despite the tight deadline of just 2 days, I was able to create a playable game.
 
During the development of this project, I gained a lot of knowledge and experience in game development, particularly in using the Unity UI system and integrating it with the player's cursor. I also developed a better understanding of the game of chess.
 
Overall, this project was a valuable learning experience that helped me improve my skills in game development and broaden my knowledge of chess.
