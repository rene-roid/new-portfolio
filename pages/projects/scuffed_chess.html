Scuffed Anime Chess

This is a chess game made in Unity. I made this game as a side project to learn more about chess and C# in Unity. I also wanted to learn more about the game development process. I made this game in 2 days.

It was a challenge to make a chess game in 2 days, but I managed to make a playable game. I learned a lot about chess and game development. I also learned how to use the Unity UI system with the player cursor.
