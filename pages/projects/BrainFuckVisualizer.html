BrainF**k Visualizer

This is a BrainF**k visualizer and compiler made in C# using Unity.

This project allows you to visualize the execution of a BrainF**k program in real time.

I made this project because when I wanted to take on a challenge and do anything in BrainF**k I ussually had a nightmare knowing where the pointer was or what value contained each cell, so I made this project to be able to know what is happening when I run the code.

Take a look at the source code! Download the project!
