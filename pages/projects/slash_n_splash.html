My first mobile game

I created a mobile game called Slash N Splash with my group, HR Games. The game was made using Unity and is a thrilling action game where the player must slash through enemies to make them explode. The game features intense gameplay, constant movement to keep players engaged and entertained.
 
We were able to learn valuable skills in teamwork, problem-solving, and project management, which we will use in future endeavors.
 
Overall, we are proud of the final product and are excited to share it with players around the world. Slash N Splash is a testament to our passion for game development and our commitment to creating fun, engaging experiences for players.
