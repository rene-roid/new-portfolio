Rythm Runner Game

This is a rhythm runner game created using Unity and C# programming language for a game jam held at my school in 2022. The game challenges the player to avoid obstacles by dodging left, right or jumping, while keeping up with the rhythm of the background music.

During the development of this project, I gained valuable experience in synchronizing the game with the music. To achieve this, I used MIDI files, which allowed me to map each note of the music to a specific action in the game.
Overall, this project has taught me a lot and helped me improve my skills in game development.
