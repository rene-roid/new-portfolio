Rythm Runner Game

This is a rythm runner game made in Unity. It was made for a game jam in my school in 2022.

The game is made using the Unity engine and the C# programming language. The game is a rythm runner game, where the player has to avoid obstacles by dodging left, right or jumping.

I learned a lot from this project, since I had to learn how to syncronize the game with the music.
I used midi files to syncronize the game with the music, each note in the midi file is a different action in the game.
